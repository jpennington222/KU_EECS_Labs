To compile the program, type "make" then enter

To run the program type "./Lab02 data.txt" then enter

The sample file should be formatted in the following example:

    Chhetri:  72,  Ronaldo:  99,  Chitalu:  84

                      or

    Chhetri:  72
    Ronaldo:  99
    Chitalu:  84

Follow the on-screen instructions to execute the program

For the "AddPlayer" function, input the player in the format "name: score" where score is an integer

For the "RemovePlayer" function, input the player in the format "name: score" where score is an integer
